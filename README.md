#Think4 Procurement and Contracting

## Starting up

Starting up is easy. Just clone the repository, install dependencies, and run `manage.py`.

```
$ git clone git@github.com:jamiecounsell/procurement.git
$ cd procurement/
```

```
$ npm install
$ bower install
```

```
$ ./app/manage.py
```
